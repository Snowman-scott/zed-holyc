# zed-holyc

This is an extension for zed to add holyc support

## Why?

Why not?

Who does not want to use gods favourite programming language in the best IDE?!

<sub>I also got bored and was trying to program hc without syntax highlighting T_T</sub>

## Referances & base
This repo was made and based off of:
- [jamesbradfard/holyc-lang](https://github.com/Jamesbarford/holyc-lang) -> Compiler & Transpiler
- [Zuhaitz-dev/holyc.nvim](https://github.com/Zuhaitz-dev/holyc.nvim) -> Holyc support for nvim (github)
- [OverLessArtem/holyc.nvim](https://codeberg.org/OverLessArtem/holyc.nvim) -> Holyc support for nvim (Codeberg)
